# ACCEPTANCE — 任务资源配置增强

## 执行结果验证
- 打开开发服务器并进入 TaskManagement 页面。
- 点击“创建任务”：
  - 资源配额设置中“内存(GB)”默认显示 32；
  - 资源类型为 CPU 时不显示“加速卡数量”；
  - 切换资源类型到 GPU 或 NPU 时出现“加速卡数量”，默认值为 1；
  - 输入越界（例如 acceleratorCards=0、9）提交，会阻止提交并显示错误提示。
